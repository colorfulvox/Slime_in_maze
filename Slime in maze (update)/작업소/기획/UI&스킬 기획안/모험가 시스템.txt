공학자 스킬 시스템

(패시브) 전용 드론

공학자 전용 드론이 일정 시간에 재료를 랜덤으로 가져온다.
(몬스터 출현,인간 형태 제한 없이 드론은 재료를 가져온다.)
재료의 이름은 미정.
하급 50% 
중급 30%
고급 20%
재료는 총 3개로 기본적으로 확률에따라 랜덤으로 수집되며 
여기서 가짓수,각 등급별 갯수는 밸런스 시뮬레이션에서 결정된다.
* 전용 드론같은 경우 챌린지에서 얻은 성장 카드를 통해 업그레이드가 가능하며
업그레이드 수치는 타입 획득 확률 증가, 가짓수 증가, 등급별 갯수 증가
정도가 된다.

보물상자

드론은 모험가에게 복귀시 랜덤확률로 보물 상자의 위치를 네비게이션형태로 알려준다.
단, 해당 보물 상자는 실시간으로 생성되는 보물 상자이며
인게임 상에서 등장하는 Default 보물 상자는 알려주지 않는다.
드론이 생성한 보물 상자 같은경우 설계 도면,일정한 골드,추가되는 음식,아이템이 랜덤으로
등장한다.


모험가는 재료를 통해 변신 장치,스킬 제작이 가능하다.

변신 장치 : 변신 장치로 변신이 가능하며 
변신 장치의 게이지가 다 찰경우 사용이 가능하며
사용할경우 게이지는 소모된다.
챌린지 퀘스트를 통해
쿨타임,변신시간,횟수의 업그레이드가 된다.


(액티브) 스킬 : 제작
모험가의 스킬은 제작 하나뿐이며
제작 버튼을 클릭할경우 재료의 등급,갯수에따라 제작이 가능하며
제작된 아이템은 가방에 저장되어 클릭할경우 타입에따라 사용이 가능하다.

도면

모험가는 각 스킬 타입의 도면을 획득해야 해당 스킬의 사용가능하다.
도면 같은 경우 일반 퀘스트, 챌린지 퀘스트 보상 항목에서 획득이 가능하며
각 타입의 스킬 개방,업그레이드는 모험가 방의 테이블에서 진행된다.
스킬 개방 같은 경우 각 타입별로 도면 갯수는 1개로 제한하며
업그레이드는 등급에따라 도면의 갯수로 개방가능하다.
